Available Locations:

  /403
  /404
  /405
  /500
  /502
  /503
  /504
  /520
  /524
  522 - No tunnel - external link
  522 - With tunnel - external link

Other:

  /cookie - Set Cookie - returns a cookie for testing through proxy, Access, and/or cloudflared
  /public - Serve files from ./public
  /readme - The README page for this program
  /request - Print the request headers as the server sees them
  /response - Set the response headers, ex: https://www.gotestserver.com/response?header1=value1&header2=value2

The default is to return the request headers
